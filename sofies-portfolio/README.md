# Sofie Wennström | Digital Portfolio
Detta är min digitala portfolio, byggd med React, skapad för att presentera mina projekt inom systemvetenskap, webbutveckling, apputveckling och fotografi. Portfolion är designad med fokus på responsivitet och modern användarupplevelse.

## Funktioner (Features)
Här är de viktigaste delarna av min webbplats:
- Responsiv design: Fullt responsiv layout som fungerar utmärkt på alla enheter (mobil, surfplatta och desktop).
- Enkel navigering: Ren och intuitiv navbar med en responsiv hamburgermeny.
- Projektsektion: Uppdelning av projekt i kategorier (Skolprojekt, Privata projekt, Fotografprojekt och mina fotografier) för tydlig presentation.
- Teknologier: Byggd med React för effektiv komponenthantering och routing.

## Teknologier (Stack)
Här är de viktigaste verktygen och ramverken jag har använt:
- Huvudramverk: React
- Routing: React Router DOM (för att hantera navigering mellan sidorna)
- Styling CSS3 (med fokus på Flexbox och mediafrågor)
- Ikoner: Material UI Icons ( @mui/icons-material)
- Laddningshantering: Vite

## Installation och Start (Så här kör du portfolion lokalt)
Instruktioner för att klona och köra projektet
- Klona repositoryt: git clone https://github.com/SofieWenn/
- Gå till projektmappen: cd sofies-portfolio
- Installera beroenden: npm install
- Starta utvecklingsservern: npm run dev

## Projektstruktur och Navigering
Så här är min app uppbyggd
- src/App.jsx: Huvudkomponenten som hanterar routing och renderar Header/Footer.
- src/components/: Innehåller alla återanvändbara komponenter (Header, Footer, MainContent, Projekt m.m.).
- src/assets/: Lagringsplats för bilder och statiska tillgångar.
